# EcoExchange ♻️

## AI-Powered Waste-to-Reward Platform

EcoExchange is an AI-powered platform that encourages responsible waste disposal and recycling through rewards, NGO collaboration, and environmental awareness.

### Features
- AI Waste Analysis
- EcoPoints Reward System
- NGO Integration
- Religious Waste Management
- E-Waste Collection
- Multilingual Support
  - English
  - Hindi
  - German
  - Spanish
  - French
- QR Code Access
- Leaderboard System

### Tech Stack
- HTML
- CSS
- JavaScript

### SDGs Supported
- SDG 11: Sustainable Cities and Communities
- SDG 12: Responsible Consumption and Production
- SDG 13: Climate Action
